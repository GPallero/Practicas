Ejemplo sencillo de una promesa encadenada.

Los métodos '.then()', '.catch()' y '.finally()' son utilizados para asociar una acción posterior con una promesa establecida.

El método '.then()' toma hasta dos argumentos, ambos son funciones callback, uno para el caso resuelto de la promesa y otro para el caso rechazado.

Cada '.then()' devuelve un objeto 'promise' recién generado, que puede ser utilizado opcionalmente para encadenamiento.
